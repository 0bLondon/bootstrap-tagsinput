Bootstrap Tags Input

Bootstrap Tags Input is a JQuery based plugin providing a Twitter Bootstrap-like user interface for adding tags.

Examples

Default tags input
Alternative tag style in options: { tagClass: 'label label-success' }
Supports using objects as tags { itemValue: 'value', itemText: 'text', typeahead: [ { value: 1, text: 'The Netherlands' }, { value: 2, text: 'Belgium' }, { value: 3, text: 'Germany' }, { value: 4, text: 'France' }, { value: 5, text: 'Spain' } ] }
